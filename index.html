ExtremePAWN

A PAWN Editor (SA-MP)

Download this project as .zip Download this project as .exe

Welcome to ExtremePAWN official site.

ExtremePAWN is one of the best PAWN editors specially made for SA-MP.
It was made for making SA-MP scripting easier and better.

You can download ExtremePAWN by clicking the the download button above.

Features

  1. A better syntax-highlighting.
  2. An AutoCompleteMenu that is sycned to your script.
  3. Alot of tools that makes scripting easier.
  4. A better error/warning list.
  5. An ObjectExplorer that shows all the functions, stocks and defines defined in the script.
  6. A better include list.
  7. A project system.
  8. A nice movable UI.
  9. Tab system with cloning ability.
  10. Ability to split the screen to an infinite number of files opened in the same time.
  11. And alot more..

System Requirements

  • Windows 7 or Windows 8/8.1
  • 2 GB RAM or above.
  • .NET FrameWork 4.0 or above.
