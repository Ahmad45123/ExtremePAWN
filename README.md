ExtremePawn
===========

ExtremePAWN is from one of the PAWN editors.

It has many features, But from the main features are : 

	1)No more indent problems.
	
	2)Autocomplete.
	
	3)Better function list.
	
	4)Better syntax highlighting for the code.
